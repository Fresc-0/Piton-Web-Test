# Piton-Web-Test
[cite_start]Bu proje, Piton Technology teknik değerlendirme süreci kapsamında hazırlanmıştır[cite: 3, 12].

## Kullanılan Teknolojiler
- [cite_start]Test Otomasyonu: Playwright (JavaScript/TypeScript) [cite: 19, 42]
- [cite_start]Manuel Test & Raporlama: Microsoft Excel / Word [cite: 21, 25]

## Proje Yapısı ve Kurulum
1. Projeyi bilgisayarınıza indirdikten sonra terminalde `npm install` komutu ile gerekli kütüphaneleri kurun.
2. Otomasyon testlerini çalıştırmak için `npx playwright test` komutunu kullanabilirsiniz.

- [cite_start]Bölüm B için gnsmetal.com ve piton.com.tr sitelerine ait keşifçi test hata raporu[cite: 20, 23, 25].
